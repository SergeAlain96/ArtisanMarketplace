# Web (Phase 4)

Frontend stack:
- Next.js (App Router)
- Tailwind CSS

Implemented pages:
- `/` Home (artisan list)
- `/catalog` Product catalog
- `/artisan/[id]` Artisan detail + products + ratings summary
- `/login`
- `/register`
- `/dashboard/artisan`
- `/dashboard/admin`

Core UI components:
- `Navbar`
- `ArtisanCard`
- `ProductCard`
- `RatingStars`
- `ProductForm`

## Run

1. Copy `.env.example` to `.env.local`
2. `npm install`
3. `npm run dev`
